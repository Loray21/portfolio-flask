Tecnologias

Argo

Argo es una herramienta para la orquestación de aplicaciones en Kubernetes. Permite definir y ejecutar flujos de trabajo complejos y automatizados.

Helm

Helm es un gestor de paquetes para aplicaciones de Kubernetes. Facilita la instalación, actualización y gestión de aplicaciones en un clúster de Kubernetes.

Kubernetes

Kubernetes es una plataforma de orquestación de contenedores que permite administrar y escalar aplicaciones en un entorno de contenedorizado.

Python

Python es un lenguaje de programación versátil y de alto nivel. Es ampliamente utilizado en el desarrollo web, científico, de inteligencia artificial y más.

Django

Django es un framework de desarrollo web de alto nivel y de código abierto, basado en Python. Permite construir aplicaciones web de manera rápida y segura.

GitHub

GitHub es una plataforma de alojamiento y colaboración de código fuente. Permite a los desarrolladores trabajar juntos y realizar un seguimiento de los cambios en el código.

Maven

Maven es una herramienta de gestión de proyectos y automatización de construcción para Java, que facilita la compilación, pruebas y gestión de dependencias mediante un archivo XML (pom.xml)

Docker

Docker es una plataforma que permite crear, desplegar y gestionar aplicaciones en contenedores, asegurando que se ejecuten de manera consistente en cualquier entorno.
